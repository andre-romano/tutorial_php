Tags HTML servem pra formatar o texto, e inserir componentes para os usuarios interagirem com a pagina.

Cada tag é aberta usando < NOME_TAG> (sem espaco) e fechada usando < /p> (sem espaco). Tudo que for colocado dentro da tag pertence a ela.

Cada tag pode ter atributos, que são colocados dentro da tag de abertura < NOME_TAG ATRIBUTO="VALOR"> (sem espaco). Os atributos modificam alguma propriedade da tag, como valor, nome, id, classe, etc. Para nos aqui, os principais atributos que precisaremos são class, id, nome, value e checked.
